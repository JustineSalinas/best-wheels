# Best Wheels Car Display

Website for Best Wheels Car Display — Iloilo City, Philippines.

Built with Next.js 14, TypeScript, and Tailwind CSS.

## Features

- 21-unit inventory with car detail modals
- Financing calculator (12–60 month terms)
- Bank partners strip (BDO, BPI, Metrobank, RCBC, EastWest, PSBank, Security Bank, UnionBank)
- Chatbot with WhatsApp handoff
- Testimonials, How It Works, Why Choose Us sections
- SEO with JSON-LD structured data
- Mobile-first, fully responsive

## Development

```bash
npm install
npm run dev
```
